# 📚 Library Management System

A console-based Library Management System built in **Python** using **Object-Oriented Programming (OOP)** principles.  
This project allows users to manage book records with features like adding, removing, searching, issuing, returning, and saving records to a file.

---

## 🚀 Features
- Add new books with details (ID, Title, Author, Genre)
- Remove books by ID
- Search books by title or author
- Issue and return books with status tracking
- Display all books in the library
- Save records to a text file for persistence

---

## 🛠️ Tech Stack
- **Language:** Python  
- **Concepts:** Object-Oriented Programming (Classes, Methods, Encapsulation)  
- **File Handling:** Text file storage for records
